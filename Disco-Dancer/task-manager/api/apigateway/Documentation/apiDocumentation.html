Introduction

This is a test API deployed for testing simple Rest api functionality with OAuth.

Use quick start guide to test this api.

 

Quick Start Guide :

   

How to consume the API

This is a Geico internal api, consumers will be accessing it from Geico internal network and/or from Azure network

Endpoints

From developers machine : harouter.agnpdp.np1.aze1.cloud.geico.net

Authentication

Authenticate using JWT token before accessing the resources.

Refer platform Teams documentation to request Access to the API documentation

 

Sample cURL getToken

        
        

        curl --location 'https://login.microsoftonline.com/7389d8c0-3607-465c-a69f-7d4426502912/oauth2/v2.0/token' \ 

        --header 'Content-Type: application/x-www-form-urlencoded' \ 

        --data-urlencode 'scope=api://apisixrestapi-apimtm-dv1-v1/.default' \ 

        --data-urlencode 'grant_type=client_credentials' \ 

        --data-urlencode 'client_id=xxxx' \ 

        --data-urlencode 'client_secret=xxxxxxx'
